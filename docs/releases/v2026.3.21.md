# v2026.3.21

**Release date:** 2026-03-02

## Bug Fixes

- **fix: handle CancelledError in CLI timeout wrappers** (#75, PR #76)
  - Nested `asyncio.wait_for` calls could leak `CancelledError` instead of `TimeoutError` in Python 3.12+
  - All CLI commands now gracefully handle both exception types
  - Added 12 parametrized tests for timeout handling

- **fix: stop auto-initializing Sentry on import** (#123, PR #77)
  - Removed `init_error_reporting()` call at module level that was capturing ALL errors in the host process (e.g., Home Assistant)
  - Added source-module filter to `before_send` — only Yarbo stack frames are reported
  - CLI entry point now explicitly initializes error reporting
